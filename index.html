Sobre mí

¡Hola! Soy Sara, tengo 24 años y estudio Desarrollo de Aplicaciones Web. Empecé en este ámbito por curiosidad y, con el tiempo, descubrí que me apasiona transformar ideas en sitios web funcionales y atractivos. Me especializo en la parte Frontend, trabajando con HTML, CSS y JavaScript para crear interfaces modernas, intuitivas y visualmente cuidadas. Me interesa especialmente combinar el diseño y la usabilidad para lograr experiencias web agradables y eficientes.
