RETROPLAY

RetroPlay es una tienda de alquiler de videojuegos retro desarrollada en equipo. En este proyecto implementamos metodologías ágiles (Scrum) y nos enfocamos en una experiencia de usuario nostálgica. Esta web es un proyecto que debemos desarrollar durante todo el primer curso así que está sometida a cambios constantes.

TECNOLOGÍAS: HTML, CSS, JS, PHP
